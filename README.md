# Trabalho-de-Álgebra-e-Geometria-computacional
Trabalho envolvendo construção de matriz para computação, onde há classes Álgebra Linear, Matrix e Vector

Construção de uma matriz para linguagem computacional.

Essa estrutura envolve Classes Vector, Matrix e Linear Álgebra

# A classe Vector terá dois métodos 

# Get: Que vai acessar um valor específico do vetor que está em uma posição determinada 

# Set: Vai alterar o valor, onde recebe o valor 'i' e o novo valor que vou inserir nele, permitindo que evite colocar dados que não existe
naquela posição.

# A classe Matrix terá dois métodos 

# Get: Vai entrar naquela estante de dados e retirar o valor exato que está na coluna e linha que pediu.

# Set: Coordenar e trocar o valor que está por um novo, permitindo alterar, sem a necessidade de recriar a matriz toda 

# A classe Álgebra Linear

# Transpose: Transformar o vetor linha em uma matriz coluna, muito importante para cálculos que a orientação do vetor importa.

# Times: Multiplicar a matriz

# Sum: vai unir as duas estruturas do mesmo tipo 

# Gauss: Vai preparar o sistema para ser resolvido. Ele faz uma matriz complexa ser triangular

# Solve: Vai vê o valor x e y chamando o Gauss para vê se está escalonada e fará a substituição retroativa
